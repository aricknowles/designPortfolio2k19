Interactive Q investigations

An investigation in letterforms, specifically in exploring a letterform in the 3d. The project focuses on the letter “Q” in the light weight of American typewriter. This is only a small collection of the many different experiments that were done throughout. The landing page was built to allow the viewer to interact and play with the letter, in a similar way that was done over the course of developing the piece. The website was built using HTML, CSS, Javascript, and Jquery. →
